# 💼 Job Portal Web Application

A full-stack Job Portal web application where users can search and apply for jobs, and recruiters can post and manage job listings.

---

## 🚀 Features

### 👨‍💻 User Features

* 🔍 Search jobs by keyword, location, category
* 📄 View job details
* 📝 Apply for jobs
* 👤 User authentication (Login / Signup)

### 🏢 Recruiter Features

* ➕ Post new jobs
* 📋 Manage job listings
* 👥 View applicants

### ⚙️ General Features

* Responsive UI
* REST API integration
* Secure authentication
* Clean and modular code structure

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/ankitasahay8/job-portal.git
cd job-portal
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
npm start
```

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm start
```

---

## 🌐 Environment Variables

Create a `.env` file in backend:

```
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
PORT=5000
```
